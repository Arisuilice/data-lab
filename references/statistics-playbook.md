# statistics-playbook

## 1. 统计分析基本要求

任何统计结论至少包含：

- 样本量；
- 效应方向；
- 效应量；
- 不确定性或置信区间；
- p-value 如适用；
- 假设限制；
- 是否可解释为因果。

不要只报告 p-value。

## 2. 相关分析

- 数值-数值：Pearson + Spearman；
- 非线性或异常值明显：优先 Spearman；
- 多重相关时考虑校正；
- 相关不代表因果。

报告格式：

```text
X 与 Y 存在中等正相关（Pearson r=..., Spearman ρ=..., n=...），说明二者在样本中共同变化，但不能说明 X 导致 Y。
```

## 3. 组间差异

| 情况 | 方法 |
|---|---|
| 两组、近似正态 | t-test + Cohen's d |
| 两组、非正态/离群 | Mann-Whitney U + rank-biserial |
| 多组 | ANOVA / Kruskal-Wallis + post-hoc |
| 类别比例 | chi-square / Fisher + Cramer's V |

必须报告每组样本量。

## 4. 回归解释

线性回归报告：

- 系数；
- 标准误；
- 置信区间；
- p-value；
- R² / adjusted R²；
- 残差诊断；
- 共线性检查。

若目标是预测，不要把 OLS 当作唯一模型。

## 5. 交互项解释

含交互项时，禁止孤立解释主效应为固定边际效应。

若模型为：

```text
Y = β0 + β1 X + β2 Z + β3 XZ
```

则：

```text
X 的边际效应 = β1 + β3 Z
Z 的边际效应 = β2 + β3 X
```

报告必须给出至少 2-3 个代表性水平下的边际效应，或用图展示。

## 6. 因果边界

只有满足明确识别设计时才使用因果措辞，例如：

- 随机实验；
- DiD 且平行趋势可信；
- IV 且工具变量假设可信；
- RDD 且断点附近连续性可信；
- 匹配/倾向得分只能谨慎表述为调整后关联。

普通横截面回归只写“相关”“关联”“预测”。

## 7. 多重检验

若同一问题下进行多个显著性检验，应报告校正方法，例如 Holm 或 Benjamini-Hochberg。

不需要为了少量探索性 EDA 过度形式化，但正式 finding 必须克制。
